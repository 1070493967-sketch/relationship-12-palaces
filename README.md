# 关系十二宫

> **你在他心里，是第几宫？**

一套基于亲密关系十二宫格理论的AI诊断系统。帮你看清自己在关系里的位置，看懂TA的底层逻辑。

---

## 这是什么？

关系十二宫是一个WorkBuddy AI Skill（技能），安装后可以通过对话引导完成三种诊断：

| 模式 | 做什么 | 适合谁 |
|:----|:------|:------|
| **看自己** | 基于你的情感经历和行为模式，定位你在关系十二宫中的位置 | 对自己的关系模式感到困惑的人 |
| **看TA** | 输入你对TA的观察和感受，分析TA的核心驱动和关系逻辑 | 想看懂伴侣/暧昧对象的人 |
| **双人互测** | 双方各自评价自己+评价对方，输出匹配分析和偏差报告 | 情侣/夫妻，想知道"我们之间到底发生了什么" |

---

## 十二宫格是什么？

一套覆盖人类亲密关系全光谱的3×4理论矩阵。两个维度交叉：

**维度一：进入关系的能力**
- 第一层：有能力进入（拥有主流社会认可的关系资本）
- 第二层：有能力但自我限制（自己有钥匙，但不敢开门）
- 第三层：被结构性地剥夺资格（社会没给入场券）

**维度二：关系建立后的状态**
- 活的：有真实的情感流动和生命力
- 死的：外壳完整但内部枯竭
- 工具化的：关系是手段不是目的
- 从未进入：亲密关系的完全空白

两个维度交叉，构成十二个关系生态位。每个人的关系状态，都可以在十二宫中找到对应的格子。

**十二宫概览：**
①绽放者/守夜人 · ②守墓人 · ③猎手/交易师 · ④（逻辑空白）
⑤例外者 · ⑥自我囚禁者 · ⑦孤芳的掠夺者 · ⑧孤岛
⑨围墙内的春天 · ⑩被遗忘的共存 · ⑪被围猎的羔羊 · ⑫沉默的星

（完整的理论体系说明见项目文档）

## 安装与使用

### 前置条件

你需要安装 [WorkBuddy](https://www.codebuddy.cn)（CodeBuddy的AI助手）。

### 安装步骤

1. 将本仓库的 `SKILL.md` 文件放入你的WorkBuddy skills目录：
   - Windows: `%USERPROFILE%\.workbuddy\skills\关系十二宫\`
   - macOS/Linux: `~/.workbuddy/skills/关系十二宫/`
2. 重启WorkBuddy或在对话中输入 `/skills` 确认技能已加载。
3. 在对话中输入相关话题（如"帮我看看我在关系里是什么位置"），技能会自动激活。

### 使用示例

```
你：帮我看看我男朋友是什么类型的
AI：[引导你描述TA的行为模式，然后输出诊断报告]
```

```
你：我和女朋友想一起测一下我们的关系
AI：[引导双方分别输入自评和他评，输出双人匹配报告]
```

```
你：我老是遇到渣男，帮我看看为什么
AI：[分析你的模式，定位你在十二宫中的位置，指出核心风险]
```

## 与话疗坐标镜的关系

**话疗坐标镜**（文昌星）帮你看清自己的认知位置——你卡在哪、盲区在哪、第一步往哪走。

**关系十二宫**帮你看清自己在关系里的生态位——你是什么类型、TA是什么类型、你们之间发生了什么。

两个工具是平行关系，可以独立使用，也可以配合使用（先认识你自己，再认识关系中的你）。
